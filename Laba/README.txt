Описание программы
------------------
Программа предназначена для работы с файлом списка студентов и файлом их паролей.
Вся работа выполняется через консольное меню.

Пользователь может:
- Создавать или открывать .txt файлы.
- Добавлять фамилии студентов.
- Добавлять пароль студентов.
- Указывать номер группы и номер студента.

Программа полностью управляется вводом пользователя.


Главное меню
------------
После запуска отображается меню:

1) Ввод фамилии студента
2) Ввод пароля студента
0) Закрыть программу

Выберите нужный пункт, введя соответствующую цифру.


Работа с файлами
----------------
При выборе любого режима программа:
- Показывает все доступные .txt файлы в папке myfiles текущей директории.
- Просит ввести имя файла.
- Проверяет имя (разрешены латиница, цифры, '_', '.', '-').
- Если расширение .txt не указано, программа добавит его автоматически.
- Создаёт или открывает файл в режиме дозаписи.


Ввод фамилии студента
---------------------
Программа запросит фамилию студента.

Правила ввода:
- Разрешены буквы (русские или латинские), а также символы: '-', '_', '''.
- Фамилия должна начинаться с заглавной буквы.
- При неверном вводе программа предложит исправление.
- Введите '0' для выхода в меню.

После фамилии вводится номер группы и номер студента.


Ввод группы
-----------
Формат группы: M7O-206BV-24  
Строка проходит специальную проверку.

Дополнительно:
- '_' — поле оставляется пустым.
- '0' — возврат назад.


Ввод номера студента
--------------------
Введите положительное целое число.
Можно ввести '_' для пустого значения.


Ввод пароля студента
--------------------
Пароль должен соответствовать требованиям:

- Минимум 8 символов.
- Максимум 20 символов.
- Не менее:
    * одной заглавной буквы,
    * одной строчной буквы,
    * одной цифры,
    * одного специального символа.

Если пароль начинается с пробела, включается аварийный режим —
можно сохранить любую строку.

При ошибках ввод сопровождается подсказками:
- Пароль должен содержать заглавные буквы
- Пароль должен содержать цифру
- Пароль должен содержать специальный знак
и т.д.

Для выхода в меню введите '0'.


Формат записываемых данных
--------------------------
Строки сохраняются в виде таблицы:

| Фамилия/Пароль     | Группа        | Номер |

Примеры:

| Иванов        | M7O-206BV-24 | 12 |
| PassWord!123  | M7O-206BV-24 | 12 |


Системные требования
--------------------
- ОС: Windows, Linux или Termux.
- Компилятор с поддержкой C++14.
- Наличие файлов: function.cpp, utility.cpp, settings.h, Laba.h и т.д.


Сборка и запуск
---------------
Команда сборки (Linux/Termux):

g++ main.cpp function.cpp utility.cpp -std=c++17 -o Laba

Windows:

g++ main.cpp function.cpp utility.cpp -std=c++17 -o Laba.exe

Запуск:

./Laba
или
Laba.exe


Примечания
----------
- Все файлы создаются в текущей директории.
- При ошибочном вводе программа всегда сообщает причину.
- Для навигации используйте ввод цифр и символов, указанных в подсказках.
