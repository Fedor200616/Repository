================================================================================================================
Инструкция пользователя
================================================================================================================

Программа позволяет вести учет студентов: фамилии, номера групп и пароли.

1. Главное меню:
   Вы можете выбрать один из пунктов:
   1) Ввод фамилии студента
   2) Ввод пароля студента
   3) Инструкция пользователю
   0) Закрыть программу

2. Для ввода фамилий студентов нажмите 1 в главном меню:
   - Выбирите название файла с фамилиями из списка доступных или напишите свое.
   ДОПУСТИМЫ ТОЛЬКО ФАЙЛЫ ФОРМАТА .txt
   - Вводите фамилию студента. 
     * Фамилия должна начинаться с заглавной буквы.
     * Допустимы только буквы, а также символы -, _, '.
     * Введите 0 для возврата в главное меню.
   - Вводите номер группы студента в формате M7O-206BV-24.
     * Можно оставить пустым, введя _.
     * Введите 0 для возврата к выбору фамилии.
   - Вводите номер студента в группе.
     * Можно оставить пустым, введя _.
   - Подтвердите запись данных [Y/N/R].

3. ля ввода паролей студентов нажмите 2 в главном меню:
   - Выбирите название файла с паролями из списка доступных или напишите свое.
   ДОПУСТИМЫ ТОЛЬКО ФАЙЛЫ ФОРМАТА .txt
   - Введите пароль студента.
     * Пароль должен включать:
       - хотя бы одну заглавную букву,
       - хотя бы одну строчную букву,
       - хотя бы одну цифру,
       - хотя бы один специальный символ.
     * Длина пароля: 8–20 символов.
     * Введите 0 для возврата в главное меню.
   - Вводите номер группы и номер студента, как в пункте 2.
   - Подтвердите запись данных [Y/N/R].

4. Общие правила:
   - Все файлы сохраняются в папке "myfiles".
   - Все изменения добавляются в выбранный файл.
   - Программа проверяет правильность ввода и предупреждает об ошибках.
   - Для отмены действий используйте 0 или R в соответствующих пунктах.

! Ввод фамилии кирилицей доступен, но работает в экспериментальном режиме, применять с осторожностью
